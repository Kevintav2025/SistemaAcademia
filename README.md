# 🏋️‍♂️ SistemaAcademia - Gestão de Alunos e Matrículas
🔗 **Acesse o projeto online:** [https://kevintav2025.github.io/SistemaAcademia/]

Um sistema moderno e intuitivo desenvolvido para automatizar o gerenciamento de academias, estúdios de fitness ou crossfit. O projeto une a robustez da lógica em **Java** com a flexibilidade do **Google Apps Script** para criar uma interface web responsiva conectada diretamente a planilhas automatizadas.

## 🚀 Funcionalidades Principais

* **Cadastro de Alunos:** Registro completo de dados pessoais, contatos e informações físicas essenciais.
* **Controle de Planos e Mensalidades:** Gestão de matrículas ativas, modalidades (Musculação, Dança, Artes Marciais, etc.) e status de pagamento.
* **Painel de Frequência:** Monitoramento simplificado do fluxo de alunos e controle de acessos.
* **Interface Dark Mode:** Telas limpas, de alta legibilidade e focadas na experiência do usuário administrativo.

## 🛠️ Tecnologias Utilizadas

* **Lógica e POO:** Java (Estruturação de classes, herança, encapsulamento e tratamento de dados).
* **Front-end Web:** HTML5, CSS3 (Design moderno com variáveis nativas e Flexbox) e JavaScript (Manipulação de DOM).
* **Integração e Cloud:** Google Apps Script (`google.script.run`) servindo como a API de comunicação em tempo real.
* **Banco de Dados:** Google Sheets (Planilhas Google) para armazenamento seguro, centralizado e sem custos de infraestrutura.
